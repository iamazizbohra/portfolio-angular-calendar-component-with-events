Features:

  • Calendar component in Angular
  • Form to add a calendar appointment
  • Form to edit a calendar appointment
  • Ability to delete the appointment
  • Ability to move appointment between dates

Others:

  • Used shared component
  • Used ESLint for linting
  • Used Dependency Injection
  • Used Lazyload for module/routes
  • Used RXJS subject, observables, operator
  • Used Date() class to work with time/date
  • Used router, router-outlet for navigation
  • Used Angular reactive forms with validators
  • Used Angular Material for styles and Angular CDK for DnD
  • Used Ngfor structural directive to render dates and appointments
